KrishiLink is a digital platform that helps farmers access fair prices, expert guidance, weather updates, and connects them directly to buyers — eliminating middlemen.

Built for the <abbr title="Indian Farmers">*hands that feed the nation*</abbr> with the hope of a better India.

# 🌐 The Prototype
<i><b>https://krishilink.pages.dev/</b></i>

# 🎯 Vision
Farmers constitute more than 54% of the Indian population, but the support provided or the status and acknowledgment given to them for their work is quite minimal. Farmers are exploited by dealers and middlemen and often do not recieve the capital they were entitled to for their produce.

KrishiLink originated from the idea of providing farmers the capital that is rightfully theirs by eliminating middlemen and connecting them directly to their buyers.

# ⭐ Why choose KrishiLink?

## 🛒 Digital marketplace
> Direct contact between sellers and buyers

KrishiLink allows both farmers and buyers to connect seamlessly using their seperate account categories.
This feature eliminates middlemen and increases the profits of every farmer connected with us.

## 🪙 MSP Reference Guide
> Latest data provided by the *Indian Government*

Increases awareness by providing access to updated *Minimum Support Prices* (MSP) which help make informed decisions and get fair value for each farmer's produce.

## 🦾 Krishibot AI Assistant
> Powered by *Gemini 2.5*

Provides answers, personalized farming advice, crop recommendations, and real-time solutions to improve farm yields.

## 🌤️ Weather Updates and Suggestions
> Powered by *Open-meteo*

Accurate local weather predictions fetched real-time to help plan farming activities and protect harvest.

## 🐞 Crop-Health Analysis
> Powered by *Gemini 2.5*

AI-powered disease detection and expert recommendations to keep crops healthy and thriving.

# 🛠 Tech Stack

- **Frontend Framework**: React 18 with TypeScript, bundled by Vite
- **UI Components**: shadcn/ui built on Radix UI primitives
- **Styling**: Tailwind CSS with tailwind-merge and tailwindcss-animate
- **Routing**: React Router DOM v6
- **State & Data Fetching**: TanStack Query (React Query v5)
- **Forms & Validation**: React Hook Form + Zod
- **Database & Backend**: Supabase (PostgreSQL, Auth, Realtime)
- **Data Visualization**: Recharts
- **Icons**: Lucide React
- **Development & Scaffolding**: Lovable (AI-powered web app builder)
